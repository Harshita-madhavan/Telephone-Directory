A telephone directory is a listing of telephone subscribers in a geographical area or subscribers to services provided by the organization that publishes the directory. It consists
of the name as well as the phone number added as a contact in the directory, arranged in alphabetical order.
The functions that can be performed in the directory are 
❖ Add a contact
❖ Edit a contact ( edits name, number or both )
❖ Delete a contact
❖ Display contacts
Based on the choices entered, the corresponding function is executed.
